# Test Storage Deployment

This is a simple test deployment to verify Git-Ape Azure DevOps pipelines work correctly.

## Resources Created

- Resource Group: `rg-gitapetest-dev-eastus`
- Storage Account: `stgitapetestdev[uniquestring]`

## Purpose

Validates the complete Git-Ape deployment pipeline including:
- Template validation
- OIDC authentication 
- ARM deployment execution
- State file generation
